# FoodBridge 🍱

FoodBridge is a smart food donation and distribution management system built using **MySQL, Flask, and REST APIs**.

It connects **donors, receivers, and administrators** through a real-time workflow for:

* food donation
* request management
* approval and fulfilment
* expiry tracking
* audit logging
* live dashboards

## Tech Stack

* Python Flask
* MySQL
* HTML / CSS / JavaScript
* REST APIs
* SQL Views, Triggers, Procedures, Events

## Features

* Normalized 3NF database schema
* Stored procedures and triggers
* automatic expiry event
* real-time dashboards
* donor and receiver modules
* admin analytics
